I used to be a Pro Tools / Logic / Reason user but since ca 2014 I try to make all my music with Free Software.

My Setup:

Software

  • Ubuntu Studio 14.04
  • Ardour - for recording instruments and mixing
  • LMMS - For playing keyboards via MIDI
  • Hydrogen - for creating drum patterns
  • Jack - connect all the things

Hardware

  1. M-Box 2 - Bought this in 2005, works fine for mixing and quickly recording 2 channels
  2. M-Audio keystation 49e
  3. Korg PadKontrol MIDI Pad - for finger drumming
  4. M-Audio ProFire 2424 - Connected to a PCI FireWire card with VIA chipset using FFADO drivers
  5. Focusrite Octopre MK2 for extra 8 channels via ADAT
